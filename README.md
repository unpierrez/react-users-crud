# Instalação e Configuração:
- npm install: Para instalar as dependências.
# Comandos de Desenvolvimento:
- npm run server: Para rodar o JSON server (deixar rodando a parte).
- npm run dev: Para rodar a aplicação.

- React
- React Query
- Material UI
- React Hook Form
- Context API
- Zod
- TypeScript
